# Clef

Clefff (clef for short) is the tentative name of a progamming language for writing music. 

the v0 implementation is in typescript for fast iteration.

Clefff is designed to be a language-as-a-library (like Lua) so it's easy to host. 

The v0 implementation isn't focusing on being nice-to-host yet, since we are still figuring out what the language _is_. 

See the [overview](v0/spec/01-overview.md) for more info.

---

# Spec: 

Currently the specification for the language is in the v0 directory. As elements of the language are more clearly defined and tested, specification documents will begin moving to a top level directory. Changing language features with move through an RFC process with proposals and plans for either modifying the existing spec or adding onto it (sortof like the existing workshop documents)
